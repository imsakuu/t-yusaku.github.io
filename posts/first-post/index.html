GitHub PagesとHugoでブログを作った

ブログを作った

特に理由はないけれど、思い立ったが吉日。ブログを作ってみることにした。
何を書いていくかはさっぱり決まっていない。このエントリを書くのでさえ、何を書こうか頭を悩ませているような状態だ。
とりあえず、AndroidやKotlinを中心に自分が興味のあることを書いていけたらいいかなと思っている。

Hugo

素敵なテーマがたくさんあったことと、導入からビルドまでの手軽さが気に入って使ってみることにした。 今日触り初めて、そのまま公開してしまったのでこれからじっくり向き合っていく。
今は、このテーマに頼りっきりだが、ゆくゆくは機能拡張やメンテナンスをしていきたい。

GitHub Pages

使ってみたかったから。それだけの理由でホスティングはGitHub Pagesを使うことにした。
いくらかの制約はあるが、ひっそりとブログをやっていく分にはきっと十分だと思う。

ひとまず、Hugoでビルドした./publicをgithub.ioにpushしているが、今後はGitHub Actionsで自動デプロイできるようにしたい。

今後

せっかく作ったブログなので、投稿を続けていきたいし、メンテナンスを通して少しでもWebの技術を学べたらと思う。
2つの目標を立てて、今回は終わりにしたい。
